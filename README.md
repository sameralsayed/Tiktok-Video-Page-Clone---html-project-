# 🎵 TikTok Video Page Replica: Billie Eilish

A professional, high-fidelity replica of the TikTok video watch page, optimized for all devices. 🚀

## ✨ Features
🎬 **Cinema Mode Video Player**: Full-screen optimized video viewing with auto-loop. 📺
📱 **Fully Responsive**: Seamless transitions between desktop split-screen and mobile vertical layouts. 📱
❤️ **Interactive Elements**: Functional Like toggles and Follow button state management. ⚡
🔍 **SEO Optimized**: Includes meta tags, Open Graph properties, and semantic HTML5. 🌐
🎨 **Authentic Design**: Matches TikTok's exact typography, spacing, and iconography. 🖌️
🧠 **jQuery Powered**: Lightweight scripts for smooth playback and UI interactions. ⚙️
📂 **Flat Structure**: All files placed in the root directory for easy deployment. 📁

## 🛠️ Tech Stack
* **HTML5**: Semantic structure and SVG paths. 🧱
* **CSS3**: Custom layout and responsive media queries. 💅
* **JavaScript & jQuery**: Interaction logic and event handling. 🧠
* **Bootstrap 5**: Grid system and rapid UI components. 📏
